# Dictionary
A minimalist dictionary application, where searching for a word gives structured results like :
<ul>
<li>Phonetics</li>
<li>Audio pronounciation</li>
<li>Part of speech</li>
<li>Origin of the word</li>
<li>Definitions</li>
<li>Example usages</li>
</ul>
<br>
